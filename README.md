# A股量化选股系统（v1.0.9）

> 单文件、零后端依赖的 A 股量化选股与交易信号系统。基于已定交易规则（3:1 风险回报、2% 止损 / 6% 止盈）做选股展示、成功概率评分、买入触发信号、系统回测胜率与每日简报。**仅生成信号，不接券商、不真实下单。**

## 二、核心交易规则（全部模块共用）

- **风险回报比**：3 : 1
- **止损**：入场价 − 2%
- **止盈**：入场价 + 6%
- **入场条件**：以成功概率 Top3 个股的**信号日（昨日）收盘价**为基准；**次日开盘价**相对基准波动 ≤ 2%（主板）/ ≤ 3%（创业板·科创板）才入场
- **选股排序**：按「成功概率」降序取 Top3
- **信号性质**：仅生成买卖信号，**不接券商、不真实下单**

## 三、功能清单（v1.0.0）

1. **选股评分（🔍 选股评分）**：6 维评分 → 成功概率 `winRate = min(88, 50 + 总分×0.35)`；Canvas 手绘 K 线五图引擎展示个股走势。
2. **策略概览（📋 策略概览）**：策略核心参数、交易胜率（系统回测）卡片。
3. **买入触发（📑 每日简报 内）**：以 Top3 基准 + 次日开盘容差判定「建议买入 / 不触发」。
4. **交易胜率回测**：基于 `import_final.json` 真实 K 线（已拉长至约 320 根/支，2025-03 起），按既定规则回测（设定周期 2025.7.1–2026.6.30）：全候选池 243 笔、胜率 27.2%、每笔期望 +0.09%；⑤组合层后 81 笔、期望 −0.04%。标定偏差 −2.9pp（v1.0.7 重标定：预测均值 24.4% vs 实现 27.4%）。
5. **每日简报（📑 每日简报）**：盘后简报 + 买入触发，含轻量 Markdown 渲染器（标题 / 表格 / 风控引用块 / 列表 / 代码）。
6. **仓位计算（🧮 仓位计算）**：按账户资金与止损比例计算单笔仓位。
7. **交易日志（📓 交易日志）**：信号与交易记录展示。
8. **风控看板（📊 风控看板）**：风险相关指标可视化。
9. **口令保护（gate）**：站点访问需输入口令（默认 `stock2026`）。
10. **自动化与部署**：交易日 09:30 自动化抓取次日开盘（tdx-connector）→ 写 `buy_signal.json`；每日 18:30 自动化构建并推送 GitHub Pages。

## 八、已知限制 / 后续优化方向

- **回测数据已拉长（④ 完成）**：`import_final.json` 经 `extend_history.js` 注入 tdx-connector 真实日线，每支 320 根（2025-03 起），覆盖设定 12 个月周期；但候选宇宙仍仅 7 支（未扩展真实选股池）。
- **当前策略实盘仍不可行**：全候选池胜率 27.2%（高于 3:1 打平所需 ~25.6%，但期望值仅约 0），主板 36.0% vs 科创板 8.9%；标定偏差 48.5pp（win 预测严重高估）；⑤组合层后期望转负（−0.04%）且回撤上限触发 130 次。
- **待优化**：① 复盘科创板 ATR 动态止损（胜率仅 ~7%，考虑加宽 K_ATR 或剔除双创）；② 提升信号方向质量（降低 win 预测高估）；③ 扩大真实候选池（>7 支）以提统计置信。（注：④ winRate 重标定已在 v1.0.7 完成，偏差 48.5pp→−2.9pp。）
